# 🧩 8 Puzzle Solver & Game

An interactive **8 Puzzle Game and AI Solver** built using JavaScript.  
This project demonstrates classic **Artificial Intelligence search algorithms** like BFS, DFS, A*, and Greedy Best First Search.

## 🎮 Features

- Interactive puzzle board
- AI algorithm visualization
- Step-by-step solution playback
- Hint system
- Multiple difficulty levels
- Algorithm performance comparison
- Smooth UI animations

---

## 🧠 Algorithms Implemented

| Algorithm | Description |
|----------|-------------|
| BFS | Breadth First Search – guarantees shortest path |
| DFS | Depth First Search – explores deeply first |
| A* | Optimal heuristic search using Manhattan distance |
| GBFS | Greedy Best First Search |
