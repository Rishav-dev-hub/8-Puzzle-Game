Algorithm Guide
BFS Optimal
Explores level by level. Guarantees the shortest path. Uses O(b^d) memory. Best for shallow solutions.
DFS Memory
Goes deep first. Not optimal — may find longer paths. Low memory usage. Fast but unpredictable.
A* Best
Uses heuristic f(n)=g(n)+h(n). Optimal & efficient. Manhattan distance heuristic guides search.
GBFS Fast
Uses only heuristic h(n). Very fast but not guaranteed optimal. Greedy approach.
